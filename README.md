# E-commerce Website
  
This provides with frontend and the back end of the e-commerce website.  
  
Technologies used:  
a) SQL  
b) PHP  
c) HTML  
d) CSS  
e) JavaScript  

Features:
a) login/logout.  
b) cart (add/remove/increase items).  
c) displays product with specfications.    
d) can register/sign up a new user.
